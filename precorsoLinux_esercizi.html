TIF 2023/2024: Introduzione a Linux e alla linea di comando  
 
  Esercizi  

 

Per gli utenti Windows "Aprire un terminale" deve essere inteso come aprire un terminale in WSL.

  • Aprite un terminale
  • Date il comando cd per essere sicuri di essere nella vostra cartella home
  • Verificate con pwd l'indirizzo della cartella home
  • Verificate con which curl che il programma curl sia installato. Se lo è, avrete una risposta simile a /...../bin/curl. In caso contrario la risposta sarà il solo prompt. In questo caso installate curl dando il comando
    sudo apt install curl
    Vi verrà richiesta la password.
  • Date il comando
    curl "http://personalpages.to.infn.it/~maina/test_FileSystem/test_FileSystem.tar.gz" --output FS.tar.gz
    Il comando curl scarica il file test_FileSystem.tar.gz dal server remoto e lo salva localmente come FS.tar.gz
  • Verificate con ls -al che il file sia presente nella vostra cartella home
  • Date il comando
    tar ztvf FS.tar.gz
    Il sistema vi mostra i file contenuti nell'archivio FS.tar.gz
  • Date il comando (in cosa differisce dal comando precedente?)
    tar zxvf FS.tar.gz
    I file contenuti nell'archivio FS.tar.gz vengono estratti dall'archivio e salvati localmente
  • Verificate con ls -al quali file sia presenti nella vostra cartella home
    La risposta attesa è: 
    total 99936
drwxr-xr-x    6 maina  staff       192 Sep 21 09:42 .
drwxr-xr-x+ 240 maina  staff      7680 Sep 21 09:41 ..
-rw-r--r--    1 maina  staff  51141795 Sep 21 09:24 FS.tar.gz
drwxr-xr-x   45 maina  staff      1440 Aug  9 11:52 Humour
drwxr-xr-x   42 maina  staff      1344 Aug 10 19:08 bin
drwxr-xr-x    7 maina  staff       224 Aug 10 19:09 phantom
Congratulazioni!
  • Date il comando ls bin per vedere la lista dei file nella cartella bin.
  • Usate il comando cd per spostarvi nella cartella bin.
  • Usate il comando cp per copiare il file ZZWW.frm in un file pippo.lol
  • Usate il comando man per cercare informazioni sul comando diff.
  • Usate il comando diff per verificare che ZZWW.frm e pippo.lol sono identici.
  • Date il comando echo pluto >> pippo.lol
  • Verificate usando il comando tail pippo.lol che il comando precedente abbia aggiunto "pluto" al fondo di pippo.lol
  • Usate il comando diff per verificare le differenze fra ZZWW.frm e pippo.lol
  • Cancellate pippo.lol usando il comando rm
  • Usate il comando cp per copiare la cartella form in una cartella pippo. Come mai non funziona?
  • Usate il comando cp -r per copiare la cartella form in una cartella pippo.
  • Usate il comando diff per verificare che form e pippo sono identiche.
  • Usate il comando rm per cancellare il file README.md da pippo.
  • Usate il comando diff per verificare le differenze fra form e pippo
  • Usate il comando rm per eliminare la cartella pippo. Come mai non funziona?
  • Usate il comando man per cercare informazioni sul comando rm.
  • Usate il comando rm -rf per eliminare la cartella pippo.

 

 Esercizi 2 
Questa seconda parte di esrcizi richiede i file e le cartelle scaricate in precedenza.
Se è necessario (Per esempio sui Thin-client delle aule F e G su cui tutti i file creati dall'utente vengono eliminati alla fine della sessione.) scaricate nuovamente il materiale con i comandi:
  • curl "http://personalpages.to.infn.it/~maina/test_FileSystem/test_FileSystem.tar.gz" --output FS.tar.gz
  • tar zxvf FS.tar.gz

 

  • Aprite un terminale se necessario
  • Spostatevi nella cartella home se necessario
  • Date il comando ls -al Humour per vedere la lista dei file nella cartella Humour.
  • Verificate che in Humour sia presente la cartella ScienceJokes
  • Date il comando ls -al Humour/ScienceJokes per vedere la lista dei file nella cartella Humour/ScienceJokes.
  • bash ha la capacità di completare indirizzi incompleti:
    scrivete sulla linea di comando ls -al Humour/Sc e premete il tasto Tab
    bash completa l'indirizzo. Premete Invio per eseguire il comando
  • È possibile usare ls per verificare che un certo file sia presente:
    date il comando ls Humour/ScienceJokes/ScientificMethod.jpg
    date il comando ls Humour/ScienceJokes/Scientific.jpg. Cosa succede?
  • Provate il comando ls con le opzioni -lh, -lt, -ltr su Humour/ScienceJokes
  • È possibile usare ls per vedere solo un sottoinsieme dei file: date il comando ls Humour/Ca*
    Il simbolo *, detto wildcard, significa "qualunque sequenza di caratteri", quindi Ca* vuol dire qualunque file il cui nome inizi con Ca, *pdf qualunque file che finisca con pdf, F*oo*jpg qualunque file inizi con F, contenga oo e finisca con jpg
    date il comando ls Humour/*pdf
    date il comando ls Humour/F*oo*jpg
  • bash tiene memoria degli ultimi comandi che avete immesso. Per richiamare i comandi precedenti usate le frecce verso l'alto e verso il basso. Quando il comando che vi serve compare sulla linea di comando lo potete modificare spostandovi con la frecce verso destra e sinistra. Potete cancellare o inserire qualunque carattere. Premete Invio per eseguire il nuovo comando. Provate per credere.
  • Date il comando mkdir Humour/pluto per creare la cartella pluto all'interno di Humour
  • Dalla vostra cartella home date il comando cp Humour/Ca* Humour/pluto per copiare tutti i file in Humour il cui nome inizi con Ca nella cartella Humour. Verificate con ls che l'operazione sia andata a buon fine.
  • cancellate la cartella pluto e tutti i file che contiene
  • Tornate alla cartella home
  • Date il comando find ./ -name micro
    Il comando cerca tutti i file e cartelle il cui nome è micro (-name micro). La ricerca esamina la cartella in cui sono e in tutte le cartelle sottostanti (./ è la cartella in cui sono)
  • Usate il comando find per cercare il file Makefile nella cartella bin/form
  • È possibile fornire a find l'indizzo assoluto della cartella da cui iniziare la ricerca:
    find /usr/bin -name curl
  • Anche con find è possibile usare delle wildcard. In questo caso il nome dei file da cercare va chiuso fra virgolette:
    find /usr/bin -name "cu*"
    find ./ -name "*bash*"
  • Date il comando man find e esaminate il risultato. Si, è un po' complicato
  • Nel vostro browser preferito cercate linux find. Aprite le prime pagine della risposta. Il web è una grande risorsa per tutti i problemi di programmazione. Non sempre è immediato trovare esattamente quello che serve.
  • Il comando cerca una serie di caratteri all'interno dei file.
    Dalla vostra cartella home date il comando grep lhapdf phantom/COMANDI_GCC.txt.
    Il comando mostra tutte le linee nel file phantom/COMANDI_GCC.txt che contengono lhapdf
  • I comandi di bash possono essere concatenati passando l'output del primo al secondo e così via. Per esempio nella vostra home date il comando ls -al. Il risultato può essere passato come input a grep per estrarre solo le cartelle per cui la riga contiene "drwx". Si usa il simbolo | (barra verticale):
    ls -al | grep drwx
  • Date il comando man grep e esaminate il risultato. Si, anche questo è un po' complicato
  • Nel vostro browser preferito cercate linux grep. Aprite le prime pagine della risposta.
  • Possiamo concatenare grep con se stesso per raffinare una ricerca. Date il comando grep lhapdf phantom/COMANDI_GCC.txt | grep home.
    Il comando mostra tutte le linee nel file phantom/COMANDI_GCC.txt che contengono lhapdf. Il risultato viene passato nuovamente a grep per selezionare le linee che contengono anche home.
