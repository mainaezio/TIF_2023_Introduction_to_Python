TIF 2023/2024: Introduzione a Linux e alla linea di comando  
 
  Esercizi  

 

Per gli utenti Windows "Aprire un terminale" deve essere inteso come aprire un terminale in WSL.

  • Aprite un terminale
  • Date il comando cd per essere sicuri di essere nella vostra cartella home
  • Verificate con pwd l'indirizzo della cartella home
  • Verificate con which curl che il programma curl sia installato. Se lo è avrete una risposta simile a /...../bin/curl. In caso contrario la risposta sarà il solo prompt. In questo caso installate curl dando il comando
    sudo apt install curl
    Vi verrà richiesta la password.
  • Date il comando
    curl "http://personalpages.to.infn.it/~maina/test_FileSystem/test_FileSystem.tar.gz" --output FS.tar.gz
    Il comando curl scarica il file test_FileSystem.tar.gz dal server remoto e lo salva localmente come FS.tar.gz
  • Verificate con ls -al che il file sia presente nella vostra cartella home
  • Date il comando
    tar zcvf FS.tar.gz
    Il sistema vi mostra i file contenuti nell'archivio FS.tar.gz
  • Date il comando (in cosa differisce dal comando precedente?)
    tar zxvf FS.tar.gz
    I file contenuti nell'archivio FS.tar.gz vengono estratti dall'archivio e salvati localmente
  • Verificate con ls -al quali file sia presenti nella vostra cartella home
    La risposta attesa è: 
    total 99936

drwxr-xr-x+ 240 maina  staff      7680 Sep 21 09:41 ..

  • Verificate usando il comando tail pippo.lol che il comando precedente abbia aggiunto "pluto" al fondo di pippo.lol
  • Usate il comando diff per verificare le differenze fra ZZWW.frm e pippo.lol
  • Cancellate pippo.lol usando il comando rm
